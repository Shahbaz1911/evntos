# **App Name**: EventWave

## Core Features:

- One-Click Event Page: Single-click event page creation with auto-generated URL.
- Event Details: Event page editing with image upload, text input (for descriptions), and map location integration via Google Maps link.
- Guest Registration: Auto-integrated registration form accepting guest information, designed and optimized for web.
- SEO URL: Use LLM as tool to create search engine optimized URL slugs based on the provided title
- Email Confirmation: Automated confirmation emails to registered guests.
- Guest List Manager: Simple guest list management UI for organizers to view and manage registrations.
- Registration Control: Toggle switch to enable/disable the registration form on the event page.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) to convey trust and reliability.
- Background color: Light gray (#F5F5F5), providing a neutral backdrop.
- Accent color: Vibrant orange (#FF5722), used sparingly to draw attention to key actions like registration buttons.
- Body and headline font: 'Inter', a sans-serif font, providing a clean, modern, and highly readable text.
- Clean, minimalist icons to represent event features and actions.
- Responsive, grid-based layout to adapt to different screen sizes.
- Subtle transitions and animations for a smooth user experience.
